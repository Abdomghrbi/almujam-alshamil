# المساهمة في المُعجم الشامل

أهلاً بك! شكراً لاهتمامك بالمساهمة 🎉

## طرق المساهمة

1. **إضافة كلمات** — سجّل الدخول وأضف كلمات مع تسجيلات صوتية
2. **الإشراف** — ساعد بمراجعة المحتوى المقدم من المستخدمين
3. **تطوير الكود** — Fix bugs, add features, improve performance
4. **توثيق** — حسّن الـ README والـ docs

## قواعد إضافة الكلمات

- كل كلمة جديدة تحتاج **تسجيل صوتي** للنطق
- بيانات جغرافية **إجبارية**: البلد → المحافظة → المدينة → المنطقة
- المصادر موثقة إن أمكن

## التطوير

1. Fork الـ repo
2. اعمل branch: `git checkout -b feature/اسم-الميزة`
3. Commit: `git commit -m "أضفت كذا"`
4. Push: `git push origin feature/اسم-الميزة`
5. افتح Pull Request

## المعايير

- Arabic comments في الكود حيثما أمكن
- ESLint + Prettier
- Unit tests للميزات الجديدة
